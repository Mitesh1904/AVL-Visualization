## local setup

1. fork/clone the repo.
2. open code in any code editor(vs code)
3. Install python in system
4. Install pip
5. Install Tkinter in system
6. Install bellow python pakeges
   ```
   pip install pyautogui
   ```
7. run the app
   ```
   python GUI.py
   ```
